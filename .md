РАБОТА №1: ДВОИЧНОЕ ПРЕДСТАВЛЕНИЕ ДАННЫХ
Цель: Изучить, как данные хранятся в памяти, и научиться работать с битами.
Выполнено:
1. Определение размеров типов данных в памяти (int, float, double, char и др.).
2. Отображение двоичного представления чисел:
   · Целые числа с выделением бита знака.
   · Числа с плавающей точкой по стандарту IEEE 754.
3. Реализация функции для изменения любого бита числа.
4. Решение задачи на инверсию знака без использования умножения.





РАБОТА №2: ОДНОМЕРНЫЕ МАССИВЫ И АЛГОРИТМ КАДАНА
Цель:  Освоить работу с массивами и реализовать алгоритм Кадана.

Выполнено:
1. Создание массива из 100 случайных чисел в заданном диапазоне.
2. Поиск минимального, максимального значений и вычисление среднего.
3. Подсчёт элементов массива, удовлетворяющих условиям пользователя.
4. Реализация алгоритма Кадана для поиска подмассива с максимальной суммой.

РАБОТА №3: ДВУМЕРНЫЕ МАССИВЫ И УКАЗАТЕЛИ
Цель: Научиться работать с матрицами исключительно через арифметику указателей.

Выполнено:
1. Заполнение квадратной матрицы по заданным схемам (спираль, зигзаг).
2. Преобразование матриц: вращение границ, обмен диагоналей.
3. Сортировка элементов матрицы с использованием алгоритма Shaker sort.
4. Выполнение арифметических операций над всеми элементами матрицы.


РАБОТА №4: ОБРАБОТКА ТЕКСТА
Цель: Освоить методы работы со строками и текстовыми данными.
Выполнено:
1. Очистка входного текста от лишних пробелов и знаков препинания.
2. Разделение текста на слова с приведением к единому регистру.
3. Фильтрация слов, не содержащих цифр.
4. Преобразование слов путём перемещения цифр в их конец.
5. Реализация алгоритма поиска подстрок в тексте.

ЗАКЛЮЧЕНИЕ:
Выполненные работы охватывают ключевые аспекты программирования на C++: от основ хранения данных и работы с памятью до сложных алгоритмов обработки массивов и текста. Все программы полностью функциональны, содержат необходимые комментарии и соответствуют поставленным задачам.

Работы готовы к проверке.
